# Enclosure Design

FreeCAD designs for the PC & Light Controller enclosure.

## Files

- `freecad/` - FreeCAD project files (.FCStd)
- `stl/` - 3D printable STL files
- `step/` - STEP files for CNC/sharing
- `docs/` - Assembly instructions, dimensions

## Design Requirements

- Mount for custom PCB
- Cutouts for:
  - Power input connector
  - 3x LED output connectors
  - Motor output connector
  - USB programming port (ESP32)
- Ventilation for components
- Mounting holes for installation
